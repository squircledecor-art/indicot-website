# Indiacot Website

Static website for [Indiacot](https://indiacot.com) — manufacturer of foiling, screen printing, foil fusing, UV coating and lamination machines.

## Structure

```
/
├── index.html          # Homepage
├── styles.css          # Main stylesheet
├── cinematic.css       # Cinematic hero styles
├── cinematic.js        # Cinematic hero script
├── tilt.js             # Tilt effect script
├── robots.txt
├── sitemap.xml
├── assets/             # Images and videos
└── products/           # Product detail pages
    ├── i-foil-online.html
    ├── i-foil-semi-automatic.html
    ├── foil-fusing.html
    ├── comp-foil-fusing.html
    ├── screen-printing.html
    ├── lamination.html
    ├── gst1060-cold-foil.html
    └── product.js
```

## Hosting

Deployed via **GitHub Pages** from the `main` branch root.

To enable: `Settings → Pages → Source → Deploy from branch → main → / (root)`
